# 🚀 Feature Requests

รายการฟีเจอร์ที่น่าสนใจสำหรับอนาคต (ยังไม่ได้พัฒนา)

## 🌟 Top Priority (ทำได้ไม่ยาก)

### 1. 🌙 Dark Mode
- **Description:** โหมดมืดสำหรับใช้งานกลางคืน
- **Benefit:** สบายตา ประหยัดแบตเตอรี่
- **Complexity:** Low

### 2. 📤 Export Markdown/PDF
- **Description:** ส่งออก knowledge เป็นไฟล์ Markdown หรือ PDF
- **Benefit:** แชร์/backup ข้อมูลได้ง่าย
- **Complexity:** Medium

### 3. 🔌 Browser Extension
- **Description:** Extension สำหรับ Chrome/Firefox เก็บ link ได้เร็ว (like Pocket)
- **Benefit:** เก็บ knowledge จาก web ได้ทันที
- **Complexity:** Medium

### 4. 📋 Templates
- **Description:** Template สำเร็จรูป เช่น "Meeting Notes", "Code Snippet", "Book Summary"
- **Benefit:** เริ่มต้นเร็ว ไม่ต้องคิด structure เอง
- **Complexity:** Low

---

## 🚀 Advanced Features

### 5. 🔗 Relations/Backlinks
- **Description:** เชื่อมโยง knowledge กันเหมือน Obsidian (bi-directional links)
- **Benefit:** สร้าง knowledge graph, หา connection ระหว่าง ideas
- **Complexity:** High

### 6. 🤖 AI Auto-tagging
- **Description:** AI วิเคราะห์ content และแนะนำ tags อัตโนมัติ
- **Benefit:** ไม่ต้องคิด tags เอง, จัดระเบียบอัตโนมัติ
- **Complexity:** High (ต้องใช้ AI model)

### 7. ⏰ Reminders
- **Description:** ตั้งเตือนความจำสำหรับ knowledge ที่สำคัญ (spaced repetition)
- **Benefit:** ช่วยจำ, ทบทวน knowledge ได้ตาม schedule
- **Complexity:** Medium

### 8. 🔄 Version History
- **Description:** เก็บประวัติการแก้ไขย้อนหลัง (undo/redo + history)
- **Benefit:** กู้คืนได้ถ้าลบ/แก้ผิด, ดู evolution ของ knowledge
- **Complexity:** Medium

### 9. ☁️ Cloud Sync
- **Description:** Sync ข้อมูลข้าม device ผ่าน Google Drive, Dropbox, หรือ self-hosted
- **Benefit:** ใช้งานได้ทุก device, backup อัตโนมัติ
- **Complexity:** High

---

## 📱 Mobile & Integration

### 10. 📱 PWA (Progressive Web App)
- **Description:** ใช้งานบนมือถือเหมือน native app (add to home screen, offline)
- **Benefit:** ไม่ต้อง install จาก app store
- **Complexity:** Medium

### 11. 📥 Import from Notion/Obsidian
- **Description:** ย้ายข้อมูลมาจาก tool อื่น (Notion, Obsidian, Evernote)
- **Benefit:** ไม่ต้องเริ่มต้นใหม่, migrate ได้ง่าย
- **Complexity:** Medium

### 12. 🔔 Webhook/API Integration
- **Description:** REST API + Webhook ให้ service อื่น connect เข้าได้ (Zapier, n8n)
- **Benefit:** Integrate กับ workflow อื่นได้
- **Complexity:** Medium
